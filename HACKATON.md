# Хакатон

**Название:** ИИ для образования: автоматизация рутинных задач

**Организаторы:** СберОбразование + Школа 21

**Кейс №3:** Генератор раздаточных материалов к уроку

## Постановка задачи

Современный учитель регулярно готовит раздаточные материалы к занятиям: карточки с индивидуальными заданиями, рабочие листы, памятки, схемы, таблицы для заполнения, тексты с пропусками и др. Это один из самых трудоёмких элементов подготовки к уроку, поскольку требуется не только придумать содержательное наполнение, но и оформить его в удобном для печати виде с учётом возраста учеников, уровня сложности и временных рамок этапа урока.

Часто раздаточный материал создаётся в последний момент «под копирку» — теряется разнообразие форм работы и заданий, страдает дифференциация. У учителя уже может быть готовый план урока с активностями и раздаточным материалом для них, но создание готовых к печати листов требует времени на ручную работу. Если делать это с помощью LLM, освободится время на проверку работ, подготовку к следующим темам и профессиональное развитие.

**Цель:** Спроектировать собственный сервис, который на основе заданной темы и опционально загруженного плана урока с использованием LLM генерирует комплект раздаточных материалов, подбирая тип раздатки под этапы урока и обеспечивая дифференциацию по сложности. Пример реализации: генерация, рабочих листов, памяток, бланков рефлексии и т. п.

## Ожидаемый результат

**Готовый сервис для решения поставленной задачи.**

**Пример образа решения:** пользователь задает контекст генерации (вручную загружает план урока или выбирает план урока из числа ранее созданных в сервисе).

Результат генерации — структурированный комплект раздаточных материалов, привязанный к этапам урока: рабочие листы с пошаговыми инструкциями, таблицы для заполнения, схемы с пропущенными элементами, памятки с опорным конспектом темы, бланки для рефлексии и самооценки и т. п. Пользователь может просмотреть комплект в виде превью, отредактировать любой материал: изменить текст заданий, поменять уровень сложности, скорректировать оформление, добавить или удалить элемент комплекта. Выгружает готовый комплект для печати или сохраняет для последующей доработки и использования.
