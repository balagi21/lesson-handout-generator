# Сравнение с существующими решениями

## Новые уроки

[«Новые уроки»](https://newuroki.net/) — это библиотека готовых учебно-методических материалов для учителей. Платформа предлагает конспекты уроков с технологическими картами, рабочие листы, презентации, шаблоны документов (характеристики, отчёты, анализы воспитательной работы) и руководства по аттестации педагогов. Все материалы структурированы по требованиям ФГОС и предназначены для использования «как есть» или с минимальной адаптацией под конкретный класс.

### Преимущества сервиса

1. Экономия времени учителя – не нужно разрабатывать документы и планы с нуля.
2. Соответствие ФГОС – готовые конспекты и рабочие листы формально соответствуют актуальным образовательным стандартам.
3. Широкий охват вспомогательных задач – помимо уроков, есть шаблоны для аттестации, портфолио, годовых отчётов, характеристик на учащихся.
4. Практическая ориентированность – материалы включают конкретные формулировки, чек-листы, советы по оформлению (например, «ошибки комиссии» при аттестации).

### Недостатки сервиса

1. Невозможность генерации материалов – сервис не создаёт уникальные раздаточные материалы под ваших учеников или тему. Только готовые статичные файлы.
2. Риск шаблонности – все разработки типовые, что может снижать творческую адаптацию к конкретному классу (разный уровень подготовки, особенности детей).
